# The Intitech FUTA Curriculum (100L - 300L 1st Semester)
This document maps FUTA academic courses to high-performance software engineering concepts. Use this to inject theoretical rigor into the Intitech architecture.

## 1. The Bare Metal Engine (Speed & Memory)
* **CSC203 (Computer Logic) & CSC204 (Assembly):** Von Neumann architecture, CPU registers, Karnaugh maps, bitwise operations. Used for writing nanosecond-level C# authorization flags and understanding CPU caching.
* **CSC305 (System Programming with C):** Memory management, pointers, concurrency, IPC, virtual memory. Used for zero-allocation C# patterns, heap management, and avoiding Garbage Collection pauses.
* **SEN309 (Operating Systems):** Process scheduling, I/O, threads, mutexes. Used for Hangfire background jobs and preventing Thread Pool starvation.

## 2. The Algorithmic Brain (Logic & Math)
* **SEN303 (Data Structures & Algorithms):** Big O Notation, Trees, Hash Maps. Used for converting $O(N^2)$ API bottlenecks into $O(1)$ lookups.
* **CSC307 (Discrete Structures):** Graph Theory, Sets. Used for WebRTC P2P routing paths and offline file-sharing logic.
* **CSC309 (Theory of Computation):** Finite Automata, State Machines, Regex. Used for unbreakable payment workflows, document approval states, and zero-backtracking string validation.
* **MTS203 (Linear Algebra) & MTS202 (Numerical Analysis):** Matrices, Vector Spaces. Used for Microsoft Semantic Kernel AI embeddings and error analysis.

## 3. The Enterprise Architect (Scale)
* **SEN204 & SEN305 (Software Requirements I & II):** Conceptual modeling, state rules. Used for translating messy real-world MKAN/AMSA hierarchies into strict database schemas.
* **SEN208 & SEN307 (Software Design / Construction):** Clean Architecture, Event-Driven Design, Fault Tolerance. Used for building microservices that don't crash when third-party webhooks fail.

## 4. The Intitech Edge (Business)
* **EMT301 (Entrepreneurship) & ECN214 (Economics):** SaaS pricing, fintech ledger logic, Nigerian business case studies.
